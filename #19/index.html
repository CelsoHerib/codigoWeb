HTML

HTML

HTML (HyperText Markup Language) es un lenguaje de marcado de hipertexto que se utiliza para estructurar y mostrar las páginas web. Se trata de un código que define la estructura de una página web y su contenido, como párrafos, imágenes, tablas, listas, videos, botones, entre otros. El HTML se basa en etiquetas y atributos para definir cómo se presentará el contenido en un navegador web. Los navegadores utilizan el HTML para interpretar y mostrar el contenido de las páginas web.

CSS

CSS

CSS son las siglas de Cascading Style Sheets, que en español significa "Hojas de Estilo en Cascada". Es un lenguaje de programación que se utiliza para dar estilo y diseño a las páginas web: Cambiar el color, la fuente, el tamaño y el espaciado del contenido Dividir en columnas Agregar animaciones y efectos visuales CSS se utiliza en conjunto con HTML, un lenguaje de marcado que se usa para crear el contenido de una página web. CSS separa el contenido del diseño, lo que facilita la gestión del aspecto visual de las páginas.

JAVASCRIPT

JAVASCRIPT

JavaScript es un lenguaje de programación que se utiliza para crear páginas web interactivas y dinámicas. Se usa para añadir características como: Actualizar fuentes de redes sociales, Mostrar animaciones y mapas interactivos, Validar formularios, Controlar multimedia, Agregar efectos de estilo dinámicos. JavaScript se ejecuta directamente en el navegador del usuario, por lo que los desarrolladores no necesitan recargar la página para realizar tareas.
